[role="lead"]
알려진 제한 사항은 이 제품 릴리스에서 지원하지 않거나 올바르게 상호 운용되지 않는 플랫폼, 장치 또는 기능을 식별합니다. 이러한 제한 사항을 주의 깊게 검토하십시오.

* Cloud Sync는 중국에서 지원되지 않습니다.
* 중국 외에 다음 지역에서는 Cloud Sync 데이터 브로커가 지원되지 않습니다.
+
** Azure 미국 정부
** Azure US DoD를 참조하십시오
